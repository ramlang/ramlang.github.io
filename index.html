Recent Work

Campfire is an open-source, self-hosted deploy preview automation tool and UI for collaborative communication through its syncing of comments with GitHub. It was built with React, JavaScript, TypeScript, and AWS.

  • Automatically deploys preview environments with GitHub Actions on every pull request.
  • Deploy previews are hosted on AWS infrastructure using AWS Elastic Container Service.
  • Share deploy previews with teammates via a public url configured to an application load balancer.
  • Comments on the proposed changes in Campfire will automatically sync back to GitHub.
  • Add visual context to feedback with a session replay recording of DOM events.
